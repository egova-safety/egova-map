Vue Flagwind Map

一、概述

vue flagwind map 是根据 falgwind map api设计的一套基于vue组件库，使用它你可以方便构建一套通用的地图应用平台。所支持的地图api类型主要由flagwind map决定，目前支持 arcgis 和 minemap两种，后续会陆续增加其他地图sdk支持。

二、集成说明

接下来我们以Arcgis为例，介绍vue项目如何集成该组织库。

1、安装vue-flagwind-map包

npm install vue-flagwind-map --save

2、添加地图配置

在项目入口文件 src/index.ts 中,添加如下代码：

三、代码示例

API

属性

名称 说明 数据类型 属性类型 默认值
vid 对象唯一标识 string 静态属性 必填
options 地图配置参数 any 静态属性
setting 默认为入口函数配置的信息。 any 静态属性
zoom 设置地图显示的缩放级别（可在options属性上定义）。 number 静态属性
center 设置地图显示的缩放级别（可在options属性上定义） Array 静态属性
wkid 设置地图显示的参考坐标系（可在options属性上定义） Number 静态属性

事件

事件名 说明 参数
on-build 组件构建完成回调。 (box:mapView)
onFormPoint 地图上的点转换成业务对象的经纬度（可在options属性上定义） (point: any)
onToPoint 业务对象转换成地图点对象（可在options属性上定义） (item: any)
onMapLoad 地图加载完成后调用（可在options属性上定义） ()
onZoomStart 地图缩放开始事件（可在options属性上定义） (level: number)
onZoom 地图缩放中事件（可在options属性上定义） (level: number)
onZoomEnd 地图缩放结束事件（可在options属性上定义） (level: number)
onPanStart 地图平移开始事件（可在options属性上定义） (args: EventArgs)
onPan 地图平移中事件（可在options属性上定义） (args: EventArgs)
onPanEnd 地图平移结束事件（可在options属性上定义） (args: EventArgs)
