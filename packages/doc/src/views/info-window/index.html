InfoWindow

一、概述

InfoWindow组件用于，在地图对应位置显示信息窗口的应用场景。它有两种用法，一种是依赖PointLayer组件，替换其原有的InfoWindow窗体；另一种，独立于图层，自定义窗口内容使用。
