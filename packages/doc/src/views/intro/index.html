Flagwind Map

概述

在使用之前需要定义全局配置文件，一般是vue项目入口函数时定义。如下：
setting.arcgis = { ...setting.arcgis, ...{
      arcgisApi: "http://192.168.101.39:8081/arcgis4js/library/3.21/",
      routeUrl: "",
      center: [118.573, 37.446],
      wkid: 4326,
      wkidFromApp: 4326,
      zoom: 9,
      slider: false,
      basemap: "dark-gray-vector",
      logo: false,
      sliderPosition: "bottom-left"
    }
};
MapLoader.loadCss(setting.arcgis);
MapLoader.loadScript(setting.arcgis);
