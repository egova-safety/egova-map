Heatmap Layer

一、概述

Heatmap Layer 是一个热力图组件，对应 flagwind map 的 FlagwindHeatmapLayer 类，通过对数据点进行汇聚操作，从而展示不同区域的热力分布情况。

API

属性

名称 说明 数据类型 属性类型 默认值
options 地层参数 any 静态属性
blurRadius 地图参数 number 静态属性 12
colorStops 热力颜色设置 Array 静态属性 [
    { ratio: 0, color: "rgb(255, 219, 0, 0)" },
    { ratio: 0.6, color: "rgb(250, 146, 0)" },
    { ratio: 0.85, color: "rgb(250, 73, 0)" },

事件名 说明 参数
