Point Layer

一、概述

该组件为点要素图层组件，对应 flagwind map 中的 FlagwindPointLayer类。我们可配置相关参数来实现不同点要素图层展示。该组件依附于Map组件，且不能脱离Map组件独立使用。

API

属性

名称 说明 数据类型 属性类型 默认值
vid 对象唯一标识 string 静态属性 必填
options 地层参数 any 静态属性
enableEdit 启用坐标修改功能 boolean 动态属性 false
enableCluster 启用要素聚合功能 boolean 静态属性 false
symbol 要素样式 any 静态属性 必填
requestData 是否异步请求数据，为true时，必须设置getDataList事件。 boolean 静态属性
requestStatus 是否异步请求状态，为true时，必须设置getStatusList事件。 boolean 静态属性
showInfoWindow 要素单击时，是否显示信息窗口 boolean 动态属性
showTooltip 要素悬停时，是否显示tooltip信息。 boolean 静态属性
selectMode 是否启用地图要素选择功能 number 静态属性 0（0为未启用 1为单选，2为多选）

事件

事件名 说明 参数
on-build 组件构建完成回调。 (layer:FlagwindBusinessLayer)
onLayerClick 图层要素单击事件。 (evt:any)
onCheckChanged 图层要素单击选择回调事件。 (evt: { target: Array; check: boolean; selectedItems: Array })
onPositionChanged 要素坐标变更回调事件。 (currentPoint: any, originPoint: any, item: any)
onVisibleChanged 图层可见性变换回调事件。 (isShow: boolean)
getImageUrl 自定义要素图标，它的优先级高于symbol，默认为null。 (model: any)
changeStandardModel 标准化实体转换方法。 (model: any)
getInfoWindowContext 信息弹窗方法。返回结果如：{title: "详细信息", content: "没有定制详细信息"} (mode: any)
